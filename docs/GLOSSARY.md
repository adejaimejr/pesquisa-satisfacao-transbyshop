# GLOSSARY

Termos importantes do projeto.

## Agente

Modelo ou ferramenta de IA que executa tarefas no projeto.

## AGENTS.md

Arquivo central de instrucoes para agentes de IA.

## Arquivo-Ponte

Arquivo curto usado por uma ferramenta especifica para apontar para `AGENTS.md`, sem duplicar regras.

## Consenso

Registro de debate entre modelos para chegar a uma decisao melhor quando houver duvida, conflito ou risco.

## Memoria Do Projeto

Conjunto de arquivos em `docs/` que preserva contexto, sessoes, decisoes, tarefas e criterios de qualidade.

## Sessao

Periodo de trabalho de um agente de IA em uma tarefa ou conjunto de tarefas.

## Skill

Instrucao reutilizavel para ensinar o Codex a executar um tipo de trabalho de forma consistente.
