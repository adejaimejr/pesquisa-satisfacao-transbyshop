# GEMINI.md

Gemini deve usar `AGENTS.md` como fonte central de instrucoes deste projeto.

Antes de trabalhar:

1. Leia `AGENTS.md`.
2. Siga a ordem de leitura indicada nele.
3. Use `docs/SESSION.md` para entender a ultima sessao.
4. Use `docs/CONSENSUS.md` se houver duvida, conflito ou decisao importante.

Nao duplique regras neste arquivo. Se alguma instrucao precisar mudar, atualize `AGENTS.md`.
